Step 1: Prepare the Components

Before starting assembly, make sure all your components are ready. If you plan to solder the switches, test the PCB first using software like VIA or QMK to ensure all connections are functional. If using a hot-swappable PCB, you can skip this step and test the switches after installation.

Lubricate the switches for smoother operation. Disassemble each switch and apply a thin layer of Krytox 205g0 to the stem, housing, and spring. Ensure you only lube the areas that come into contact to avoid over-lubing, which can make switches feel sluggish.

Remember, less is more when it comes to switches. I would recommend doing one with a light amount, assemble it, test it out, and then add more depending on how it feels and sounds.

Step 2: Install Stabilizers

Stabilizers are crucial for larger keys like the spacebar, Enter, and Shift. Follow these steps:

  • Insert the stabilizers into the PCB in the correct positions.
  • Lube the stabilizers generously with Krytox 205g0. Apply a thick glob to the ends of the stabilizer wires and inside the housing. This will reduce rattle and improve smoothness.
  • Perform the band-aid mod by placing small adhesive strips (cut from band-aids) on the PCB under each stabilizer housing. This dampens noise and gets rid of rattle when the stabilizers hit the PCB (Every time you press the key).

Step 3: Install Switches

Place the switches onto the PCB, ensuring the pins align with the PCB holes. Use slight pressure to push them in. They should click into place easily; if not, check that the pins are straight and aligned properly.

If using a soldered PCB, solder the switches after installation. Ensure each solder joint is clean and secure. For hot-swappable PCBs, test the switches after assembly to ensure functionality.

Step 4: Assemble the Case

With the switches installed, mount the PCB and plate into the keyboard case. Secure everything with screws and ensure proper alignment. Double-check that all components are snugly in place.

For improved sound, consider adding mods like the tape mod. Apply one or two layers of painter’s tape to the back of the PCB. This enhances the acoustic profile, making the keyboard sound more crisp without increasing volume.

Step 5: Install Keycaps

Attach the keycaps to the corresponding switches. Ensure the layout is correct and each keycap fits securely. Double-check for any missing or misplaced keycaps.

Step 6: Test and Customize

With the keyboard fully assembled, test each key to ensure it registers properly. You can do so easily using the website keyboardtester.com.

Troubleshooting and Tips

  • If a key doesn’t work, check the solder joints or ensure the switch is seated correctly.
  • Clean and maintain your keyboard regularly to prevent dust buildup.
  • Explore mods like the tape mod or PE foam mod for sound and feel improvements.
